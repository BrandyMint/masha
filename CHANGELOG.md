## Что нового в v0.12.0

### 🎯 Основные изменения с версии v0.11.0

#### ✨ Новый функционал - FIP-001: Report Unification

**🚀 Новая унифицированная команда /report**

Представляем новую мощную команду `/report` для получения отчетов по времени с гибкими параметрами:

**Основные возможности:**
- 📅 **Гибкие периоды**: today, yesterday, week, month, quarter, конкретные даты и диапазоны
- 🔍 **Умные фильтры**: по одному проекту (`project:slug`) или нескольким (`projects:slug1,slug2`)
- 📊 **Два формата**: summary (группировка по проектам) и detailed (с описаниями)
- 💡 **Интерактивная справка**: `/report help` с кнопочной навигацией
- 🔄 **Обратная совместимость**: старые команды `/day`, `/summary`, `/hours` работают и показывают подсказки

**Примеры использования:**
```
/report                           # Отчет за сегодня
/report week project:work         # Неделя по проекту work
/report month detailed            # Месяц с описаниями
/report 2025-01-01:2025-01-31     # Диапазон дат
/report week projects:work,test   # Несколько проектов
```

**Новые компоненты:**
- `ReportCommand` - унифицированная команда с полным набором параметров
- `ReportBuilder` - ядро системы для построения отчетов (периоды, фильтры, группировка)
- `ReportFormatter` - форматирование вывода (summary/detailed форматы)
- `ReportHelpFormatter` - интерактивная справочная система с навигацией

**Устаревшие команды (работают, но deprecated):**
- ⚠️ `/day` → используйте `/report today`
- ⚠️ `/summary day|week|month` → используйте `/report today|week|month`
- ⚠️ `/hours` → используйте `/report quarter`

Все устаревшие команды показывают подсказки о новом синтаксисе для упрощения перехода.

**Техническая информация:**
- Полное покрытие тестами (RSpec)
- Оптимизированные SQL запросы
- Service Objects архитектура
- Inline keyboard для навигации по справке

#### 📚 Документация
- 📝 Обновлен CLAUDE.md с полным описанием команды `/report`
- 📖 Migration Guide для перехода со старых команд
- 💬 Интерактивная справка внутри бота (`/report help`)

### 📊 Статистика релиза

- **Новых возможностей:** 1 major feature (FIP-001)
- **Новых классов:** 3 (ReportCommand, ReportBuilder, ReportFormatter, ReportHelpFormatter)
- **Устаревших команд:** 3 (/day, /summary, /hours)
- **Тестовое покрытие:** >90%
